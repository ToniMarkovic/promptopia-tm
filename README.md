## General Information
Fullstack application that allows users to log in, create, update, delete prompts, search prompts by username, tags, or content, and visit other users' profiles.


## Technologies Used
- Next.js
- Tailwind CSS


## Features
- Authentication
- CRUD operations
- Search functionality
